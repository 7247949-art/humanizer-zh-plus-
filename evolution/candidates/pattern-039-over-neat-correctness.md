# Pattern 39：过度工整正确感

## 识别方法

文章语气正确、结构平衡、段落长度接近，几乎没有多余停顿、插入判断或自然转向。它不一定有明显套话，但整体像被模板修平了。

## 常见触发词/结构

- 连续使用"一方面 / 另一方面 / 同时"
- 每个小节都长度接近
- 每段都以正向总结收尾
- 多个并列项权重完全一致
- 句子都保持同一种正式语气

## 为什么像 AI 写的

真实作者写作时，信息密度和情绪关注点通常不平均。AI 初稿更容易把所有内容修成同样正确、同样平稳、同样不冒犯的样子。

## 改写原则

- 打破假平衡，保留真正重要的主次。
- 允许短句单独成段，用来落判断或转向。
- 删除机械并列，把不等重的信息改成自然主线。
- 保留符合文体的作者语气，但不为了"人味"变得油滑。
